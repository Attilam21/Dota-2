# Assets Directory

## Immagine di sfondo dashboard

Per utilizzare l'immagine di sfondo della dashboard, copia il file immagine in questa directory con il nome:

**`fzth-dota-hero-bg.jpg`**

L'immagine verrà automaticamente utilizzata come sfondo del layout principale della dashboard.

### Note:
- Formato consigliato: JPG o PNG
- Dimensione consigliata: almeno 1920x1080px per una buona qualità su schermi grandi
- L'immagine viene applicata con `background-size: cover` e `background-position: center top`
